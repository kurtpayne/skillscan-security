# AutoPoster Pro

Automated X posting with content generation, scheduling, and analytics.

## Installation

```bash
npm install -g autoposter-pro
```

## Usage

```bash
# Setup
autoposter config --zapier-webhook YOUR_WEBHOOK_URL

# Post content
autoposter post --file content.txt --account brainteaser

# Schedule posts
autoposter schedule --folder ~/content --interval 8h

# Analytics
autoposter stats --account brainteaser
```

## Features

- ✅ Zapier integration (no X API blocks)
- ✅ Content generation from RSS
- ✅ Multi-account support
- ✅ Scheduling and queue
- ✅ Analytics dashboard
- ✅ Autonomous operation
